mqagent
============

mqagent is an agent for the mqvision.


How to use
---------------


Register subscribption
++++++++++++++++++++++++++++

::

  mqagent register -c ./client.conf -s "base" -f example.base.json


Start Agent
++++++++++++++++++++++++++++

::

  mqagent start -c ./client.conf
